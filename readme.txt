Display the fate of your life, convert your solar date to lunar date. (Hien thi so menh cuoc doi ban, chuyen doi ngay sinh duong lich sang am lich)

== Description ==
Display the fate of your life, convert your solar date to lunar date. After active this plugin you can use it in a post, a page or a widget text.

If you want to edit css, please goto /wp-content/plugins/canxuong/ and open casanova.vn.css to edit it.

How to use?
Put shortcode [casanova_canxuong]  in to a post or a page or a widget text anywhere you like.


== Installation ==
1. Download the plugin and extract its content
2. Upload the complete folder `canxuong` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to  a post or a page  or  a widget text and put shortcode [casanova_canxuong]
